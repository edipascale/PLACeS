Detailed Description

Defines a Popularity Distribution which follows the model described in [1].

This is used to generate requests that match those of a User Generated Content website (specifically YouTube). In short, each element is assigned a peek popularity week; the number of views it receives each week only depends on whether it is before, at or after this peek popularity week.

[1] Y. Borghol, S. Mitra, S. Ardon, N. Carlsson, D. Eager, and A. Mahanti, “Characterizing and modelling popularity of user-generated videos,” Perform. Eval., vol. 68, no. 11, pp. 1037–1055, Nov. 2011.
